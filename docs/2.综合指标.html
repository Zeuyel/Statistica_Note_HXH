3  综合指标

采用统计指标概括和分析统计总体数量特征和数量关系的方法，了解相关指标的应用和指标体系的应用。

A --> 相对指标

总量指标 --> 统计整理的结果统计分析的起点
相对指标 --> 统计整理的结果统计分析的起点
平均指标 --> 统计整理的结果统计分析的起点

3.1 总量指标

反映社会经济现象一定时间、地点、条件下的总规模、总水平的统计指标。 表现为绝对数、有名数。

国民经济宏观管理和企业经济核算的基础性指标，是计算相对指标和平均指标的基础。

  1. 同类总体
  2. 统计口径一致
  3. 计量单位一致

3.2 相对指标

两个有联系的指标对比的比值，反映事物的数量特征和数量关系。

它可以是绝对数之比，也可以是相对数或平均数之比。

A --> 动态相对指标

3.2.1 计划完成相对指标

基本公式如下

\(计划完成相对数 = \frac{实际完成数}{计划数} \times 100%\)

  1. 实际数和计划数本身为总量指标、相对指标或平均指标 ，则与基本公式相同
  2. 当实际数和计划数用提高或降低的百分数表示时：
    • 越高越好： \(计划完成相对数 = \frac{1 + 实际完成数}{1 + 计划完成数} \times 100%\)
    • 越低越好： \(计划完成相对数 = \frac{1 - 实际完成数}{1 - 计划完成数} \times 100%\)

🌰：某企业计划本年度销售利润额比上年增长20%，实际增长50%；产品单位成本比上年降低10%，实际降低7%，则：

\[销售利润计划完成程度 = \frac{1 + 50\%}{1 + 20\% } \times 100\% = 125\%\] \[单位成本计划完成程度 = \frac{1 - 7\%}{1 - 10\% } \times 100\% = 103.33\%\]

表明该企业销售利润额比计划多完成25%（越高越好），而单位成本差3.33%未完成计划

  1. 计划执行进度（计划期未结束） \(计划执行进度 = \frac{累计完成数}{全全期完成数} \times 100\%\)

  2. 长期计划的检查（计划期已结束）

    • 水平法 \(计划执行进度 = \frac{长期计划某年达到的水平}{计划规定末年达到的水平} \times 100\%\)
    • 累计法 \(计划执行进度 = \frac{长期计划期间累计完成数}{计划规定的累计数} \times 100\%\)

3.2.2 结构相对指标

总体的结构 \[ 结构相对数 = \frac{总体部分数值}{总体全部数值} \times 100% \]

🌰：\(某班男同学比例 = \frac{男同学人数}{总人数} \times 100\%\)

3.2.3 比例相对指标

相对的比例

\[ 比例相对数 = \frac{总体中某一部分数值}{总体中 另一部分数值} \times 100% \]

🌰：\(某班男女同学比例\frac{男同学人数}{女同学人数} \times 100\%\)

3.2.4 比较相对指标

两个主体之间比较

\[ 比较相对数 = \frac{某总体某指标值}{另一总体同类指标值} \times 100% \]

多采用 相对指标或平均指标 进行静态比较

🌰：\(不同班级男同学数量比较 = \frac{某班男同学人数}{另一班级男同学人数}\times 100\%\)

3.2.5 强度相对指标

两个指标之间

\[ 比较相对数 = \frac{某一总量指标值}{另一个有联系而性质不同的总量指标} \]

分子分母是调换的，正指标和逆指标

\(\frac{某班手机数}{班级学生数} = 某班手机普及率\) \(\frac{班级学生数}{某班手机数} = 某班平均每部手机使用人数\)

3.2.6 动态相对指标

时间序列

\[ 动态相对数 = \frac{报告期水平}{基期水平} \times 100% \]

3.3 平均指标

3.3.1 数值平均数

3.3.1.1 算术平均数

💡： 1）总体范围不一致：算术平均数分子分母总体范围一致，两者存在从属关系；强度相对指标不存在标志值与各单位的对应问题； 2）强度相对指标分子分母可互换，算术平均数则不可。

\(x_i \to\) 变量值

\(\bar{x} \to\) 平均数

\(f \to\) 权数（一般为次数或权数）

\(n \to\)变量个数

\(m_i \to\) 各组标志总量（i为组数）

  • 简单算术平均数 未分组资料

\[ \bar{x} = \frac{x_1 + x_2 + \cdots +x_n}{n} = \frac{\sum_{i=1}^{n} x_i}{n} \]

  • 加权算术平均数 分组资料

权数：次数、频数等，绝对数 权重：比重、比率、频率等，相对数

当然对于组距数列，变量值我们采用组中值

\[ \bar{x} = \frac{\sum xf}{\sum f} = \frac{x_1f_1 + x_2 f_2 + \cdots + x_k f_k}{f_1 +f_2 + \cdots +f_k} = \sum x \omega \]

\[ \bar{x} = \frac{\sum xf}{ \sum f} = \sum x \frac{f}{\sum f } = \sum x \omega \]

其中 \(\omega = \frac{f}{\sum f}\)

注意，加权的目的在于合理评价，所以权数的定义要有价值

⭐： 1. 算术平均数与总体单位数的乘积等于总体各单位标志值的总和。 2. 每个变量值都加或减任意数值A,则平均数也要增加或减少A. 3. 每个变量值都乘以或除以任意数值A,则平均数也乘以或除以A. 4. 各变量值与算术平均值的离差之和等于0. 5. 各变量值与算术平均值的离差平方和等于最小值

3.3.1.2 调和平均数

  • 简单 未分组 \[ \bar{x} = \frac{n}{\sum \frac{1}{x } } \]

  • 加权 分组 \[ \bar{x_h} = \frac{\sum m}{ \sum \frac{m}{x }}, \quad f = \frac{m}{x } \]

现实中由于掌握资料的限制，往往使用调和平均来作为算术平均数的变形使用： \[ \bar{x} =\frac{\sum x}{\sum f} = \frac{\sum xf}{ \sum \frac{1}{x } \cdot xf } = \frac{\sum m }{ \sum \frac{m}{x }} = \bar{x}_h, m = xf \]

🌰：例6

\(m = xf\) ，这里\(x\)为价格（题目问价格的平均），\(f\)为对应\(x\)的频数

🌰：例7

所以实际上，无论什么情况下，根据平均数基本公式，先计算出总体标志总量，再计算出总体单位总量，二者相除即可得到平均数。

💡：算术平均数与调和平均数的选择思路

1）求什么，什么是 \(x\)；

2）将计算的平均数分子分母列出；

3）已知分母不知分子，用加权，即 \(f\)； 已知分子不知分母，用调和，即 \(m\) 。

3.3.1.3 几何平均数（对数平均数）\((\bar{x}_{G})\)

适用 变量的连乘积 等于 总比率 或者 总速度的变量数列

  1. \(\bar{x} = \sqrt[n]{x_1 \cdot x_2 \cdot \cdots \cdot x_n} = \sqrt[n]{\prod_{n=1}^{\infty} x }\)

🌰：例8某企业2000一2003年产值发展速度分别为：109.6%、105.3%、110.2%和101.8%，求年平均发展速度。

\[ \bar{x}_{G} = \sqrt[n]{\pi x} = \sqrt[4]{109.6% \times 105.3% \times 110.2% \times 101.8%} = 106.7% \]

所以该企业2000一2003年产值年平均发展速度为106.7%,即每年平均递增6.7%。

  1. 加权几何平均数：分组之后 \[ \bar{x}_G = \sqrt[f_1 + f_2 + \cdots + f_n]{x_1^{f_1}\cdot x_2^{f_2}\cdots x_n^{f_n} } = \sqrt[\sum f]{\prod_{n=1}^{\infty} x^{f}} \]

3.3.2.1 众数

总体中出现次数最多的标志值

单项数列可以直接观察出，组距数列需要使用插值法来计算出

\(X_{L}、X_{U} \to 众数 组上下限\) \(\delta_{1} \to 众数 组 次数与前一组只之差\) \(\delta_{2} \to 众数 组 次数与后一组只之差\) \(d \to 众数组组距\)

\[ M_0 = X_{L} + \frac{\delta_1}{\delta_1+\delta_2 }\times d \\ M_1 = X_{U} - \frac{\delta_2}{delta_1+\delta_2 }\times d \]

🌰：组距数列计算众数

\[ m_0 = x_l + \frac{\delta_1}{\delta_1+\delta_2 } \times d = 70 + \frac{(8-5)}{(8-5)+(8-6) } \times 10 = 76 \]
